# DMS

A simple document management system with no name yet.

This is a work in progress. I wanted something very lightweight that:

- Runs on an existing folder structure, rather than import and build its own
- Indexes plaintext, PDF, images
- Does OCR on PDFs
- Allows me to do full-text search
- Supports browsing and tagging documents
- Is easily self-hostable

The scope is explicitly kept small. I just want to be able to find a document when I need it, and prepare all my tax documentation in one place.
